Training Recipe Studio

A simple interface for choosing shapes, naming a recipe, setting copy density, and adding that recipe into the training pipeline without making the user think in ML terms.

What this mockup is testing
Can a non-ML user build a training profile in under 2 minutes?
1. Pick shapes visually
2. Choose how dense the jobs should be
3. Decide hole and rotation rules
4. Add the recipe to the overnight mix
